**Projeto Integrador – Website Crocodilo Lee**

Sobre o Projeto:

Este projeto foi desenvolvido como parte da disciplina Projeto Integrador do curso de Tecnologia em Sistemas para Internet (TSI) do Instituto Federal de Brasília (IFB).

O objetivo do projeto é desenvolver um website institucional e de portfólio para a profissional Crocodilo Lee (Camila Lee Torres Eufrásio), maquiadora e cabeleireira que atua nos segmentos de noivas, beleza social e produções audiovisuais.

O site tem como finalidade centralizar o portfólio da profissional, fortalecer sua presença digital e facilitar o contato com clientes, apresentando seus trabalhos de forma organizada e visualmente atrativa.
